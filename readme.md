**Python Телеграм бот**

Для запуска бота вам необходимо установить Python 3.8+ версии (https://www.python.org/downloads/)

*Windows:*
1. `py -m venv venv`
2. `./venv/Script/activate`
3. `py main.py`

*Ubuntu:*
1. `python3 -m venv venv`
2. `source venv/Script/activate`
3. `python3 main.py`
